# Contributing to Tabquest

We welcome contributions from the open-source community! Whether you want to fix a bug, improve documentation, or propose a new feature, here is how you can help.

## Development Setup

Tabquest is structured as a pnpm monorepo. 

### Prerequisites
- Node.js >= 18
- [pnpm](https://pnpm.io/installation) >= 9

### Getting Started

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally: 
   ```bash
   git clone https://github.com/your-username/tabquest.git
   cd tabquest
   ```
3. **Install dependencies**:
   ```bash
   pnpm install
   ```
4. **Run the extension in development mode**:
   ```bash
   pnpm dev:extension
   ```

### Making Changes
1. **Create a new branch** for your feature or bugfix: `git checkout -b feature-name`
2. **Make your changes** and commit them with descriptive messages.
3. **Push to your fork** and submit a **Pull Request** to the `main` branch.

Please ensure your code follows the existing style and that you are not committing any sensitive API keys or personal data.
